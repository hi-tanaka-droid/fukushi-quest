# 福祉クエスト 〜社会福祉原論A〜

社会福祉原論A(第2〜13回)の内容で戦う、レトロRPG風クイズバトルゲームです。
PWA(ホーム画面に追加できるWebアプリ)対応・オフライン動作・進行度自動セーブ。

## 公開手順(GitHub Pages・所要5分)

1. GitHubで新しいリポジトリを作る(例: `fukushi-quest`、**Public**)
2. このフォルダの中身**6ファイル**をすべてアップロード
   (リポジトリ画面 → **Add file → Upload files** → ドラッグ&ドロップ → Commit)
   - index.html / manifest.webmanifest / sw.js / icon-192.png / icon-512.png / apple-touch-icon.png
3. リポジトリの **Settings → Pages** → Branch を `main`、フォルダを `/(root)` にして **Save**
4. 1〜2分待つと公開URLができる:
   `https://<ユーザー名>.github.io/fukushi-quest/`
5. このURLのQRコードを**一度だけ**スライドに載せる

## 学生への案内(スライドに載せる文言の例)

> **QRを読むのは今日だけ!** 開いたら「ホーム画面に追加」でアプリになります。
> - **iPhone**: Safariで開く → 共有ボタン(□↑) → **ホーム画面に追加**
> - **Android**: Chromeで開く → 「アプリをインストール」の表示をタップ(出ない場合はメニュー⋮ → **ホーム画面に追加**)
>
> 以後はホーム画面の「福祉クエスト」アイコンから起動。**オフラインでも遊べて、進み具合も保存されます。**

## ゲームの更新方法

`index.html` を新しいものに差し替えてコミットするだけ。
学生側は、次にオンライン状態でアプリを開いたときに自動で最新版になります。
(オフライン時は手元のキャッシュ版で動きます)

## 補足

- 進行データは端末のブラウザ(localStorage)に保存されます。端末を変えると引き継がれません。
- 大きく更新したときは `sw.js` の `CACHE = "fukushi-quest-v1"` の数字を上げると確実に配信されます。
